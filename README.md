# Programmeerimise-projekt
Joosep Podekrati programmeerimise projekt.
Tehtud 2023 Sügissemestri programmeerimis-aine raames.
Algeline päevikupidamise programm, mis loob, loeb, kustutab ja näitab kõiki sissekandeid. Lisaks on dark-mode funktsionaalsus tänu Osvald Nigolala nõudmisele.
Repos sisalduv debugfail.txt peaks olema nähtav, kui vajutada nuppu loetle failid.
Kõikides käskudes tuleb anda failinimi kujul *.txt, et need korrektselt avaneks. Muude failitüüpide korral ei pruugi programm töötada.
Kui on soov muuta dark-mode või light-mode välimust, siis faili lahti tehes on nende väärtused sõnedena kirjas ning lihtsasti muudetavad. 
Juhul, kui on tahtmine muuta programmi suurust, on programmifaili alguses muutujad a ja b, mis mõjutavad resolutsiooni.
